# Pokémon Card Generator

A React-based Pokémon Card Generator that fetches data from the PokéAPI and displays a styled Pokémon card with types, moves, and sprite — with the ability to download the card as an image using html-to-image.

## Features

- Generate a random Pokémon
- Dynamic gradient styling based on Pokémon type
- Displays:
  - Name
  - Sprite
  - Types
  - Moves (with power)
- Download card as PNG
- Styled using Tailwind CSS
- Uses `html-to-image` for DOM-to-image conversion

---

## Tech Stack

- React
- Tailwind CSS
- html-to-image
- PokéAPI

---

## Installation

Clone the repository:

```bash
git clone https://github.com/jhnabuel/pokedex-app.git
cd pokedex-app
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```
